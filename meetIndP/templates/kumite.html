Basic punches:

Name - Jab

Description - A fast punch that is usually used to keep safe distance between you and your opponent


Name - Body punch

Description - A punch that is aiming weak spots of the center of the body aiming to the liver and the stomach




Basic kicks:

Name - Low kick

Description - A kick that aims to the leg of the opponent


Name - High kick

Description - A kick that aims to the head of your opponent
